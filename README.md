# backStageManagementSystem

## 技术栈
vue3+mysql+node.js

## 实现功能：
①用户模块：实现对企业用户的人员变动的管理
②综合模块：实现可视化的数据实时展现最新数据
③产品模块：对公司的产品进行浏览，修改数据等功能
④日志模块：对登录系统的信息进行记录，提高信息安全能力
⑤登录模块：对想进入系统的用户进行权限设置，登录后才可操作系统

# 启动

## 前端
```yarn```

```yarn dev```

使用了node-sass，如果出现兼容性问题可去掉这个依赖，不会对项目运行产生影响

## 后端
```node app```
